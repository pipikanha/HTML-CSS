Principais formatações

Negrito (Destaque)

Nesta frase temos um termo em negrito usando a tag <b> (tag não semantica)

Nesta frase temos um termo em destaque usando a tag <strong> (tag semantica)

Itálico (Ênfase)

Nesta frase temos um termo em Itálico usando a tag <I> (não semântica)

Nesta frase temos um termo em italico usando a tag <em> (semântico)

Marcação de texto

Podemos criar também um texto marcado com a tag <mark>

Texto GRANDE & pequeno

Estamos criando um texto GRANDE e um texto pequeno, utilizando as tags <big> e <small>

Texto Deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, porém não deve ser considerado

Texto inserido

Podemos marcar um texto como inseridopara dar enfase e indicar que ele foi adicionado depois

Sublinhado

Podemos sublinhar um texto com a tag <u> (não semantica)

Texto subscrito

para inserir um texto suscrito como: H20, podemos fazer isso utilizando a tag <sub>

Texto Sobreescrito

para inserir texto sobreescrito como: x2+3, podemos fazer isso utilizando a tag <sup>

Código Fonte

O comando document.getElementById("") é um escrito em linguagem JavaScript, podemos usar essa formatação utilizando a tag <code>

Citações

Como diria o pai de um amigo: O computador é um burro muito rápido, para fazer essa citação utilizamos a tag <q>

Abreviações

A sigla GPU, é uma abreviação, e para fazer abreviações usamos a tag <abbr>

Texto Invertido

rtl, todo texto inverte da direita pra esquerda

ltr, todo texto inverte da esquerda pra direita
